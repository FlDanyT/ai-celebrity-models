<h1 align="left">Usage <img src="https://github.com/FlDanyT/FlDanyT/blob/main/assets/gif/happy.gif?raw=true" width="40"/></h1>
- Choose a personality JSON file from this repository. <a href="https://github.com/FlDanyT/ai-celebrity-models/tree/main/models">folder</a>
<br>
- Upload the JSON file to your AI chat (ChatGPT, DeepSeek, or any other LLM that supports file input).
<br>
- Paste the instruction prompt provided below before starting the conversation. <a href="https://github.com/FlDanyT/ai-celebrity-models/blob/main/promt_instructions.md">text to copy</a>
<br>
- Ask questions as usual — the AI will respond using the thinking style and worldview of the selected personality.
